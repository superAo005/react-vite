# 全新技术栈管理系统

全新技术栈的后台管理系统

- React
- Recoil
- React Query
- React Hooks
- Vite
- Typescript


## 安装使用

- 获取项目代码

```
 git clone http://git.dev.sh.ctripcorp.com/risk/vite-react.git
 ```

- 安装依赖

```
 cd react-vite-admin
 ```

```
 npm install
 ```

- 运行

```
 npm dev
 ```

- 打包

```
 npm build
 ```
