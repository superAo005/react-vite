# 全新技术栈中后台管理模板

全新技术栈的后台管理系统

- React
- Recoil
- React Query
- React Hooks
- Vite
- Typescript

## 安装使用

- 获取项目代码

```
 git clone https://github.com/superAo005/react-vite
```

- 安装依赖

```
 cd react-vite-admin
```

```
 npm install
```

- 运行

```
 npm run dev
```

- 打包

```
 npm run build
```
